# 봉봉단 (Bongbongdan) 공식 홈페이지

의협을 실천하는 봉사문파, 봉봉단의 공식 웹사이트입니다.

## 🛠 Tech Stack
- **Framework**: React (Vite)
- **Styling**: Vanilla CSS, Tailwind CSS
- **Deployment**: Vercel
- **Database/Auth**: Supabase

## 🚀 Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## 🌐 SEO & Deployment
- **Domain**: [bbd.or.kr](https://bbd.or.kr)
- **SEO**: Meta tags, OG Tags, JSON-LD (NGO Schema), Sitemap.xml, Robots.txt
